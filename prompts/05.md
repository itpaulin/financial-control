# Contexto

A gestao financeira deve ser flexível e adepta a qualquer usuário, então ele deve poder criar suas próprias categorias além das iniciais e poder apagar item de categoria que já não são mais necessários. Além disso, uma distincão visual irá ajudar a deifinir o que é mais relevante no seu orcamento.

## Tarefa
- Implemente funcionalidade de adicão de categorias, quando o usuario selecionar essa opcao, ja monte o grid na UI com o input de nome da categoria focado. 
- Implemente a funcionalidade de delete de Item de uma categoria, faca essa funcionalidade utilizando o botão direito exibindo uma unica opcao Apagar, o shadcn contem esses componentes e a cor para o botao delete também. 
- Na mesma funcionalidade de apagar item, adicione um APAGAR CATEGORIA, mas nessa abra uma modal de confirmacao caso selecionada.
- Implemente a funcionalidade de colorir de forma sutíl certas categorias, utilize o agente de designer e o DESIGN.md para pensar em uma ideia de distincao, como bordas e titulo da categoria coloridos.
- Aumente o tamanho dos textos desta rota [year]/[month]. Está muito pequeno.
- Uma categoria que tem espaco sobrando dentro do grid, um clique na regiao de baixo que não tenha itens, deve triggar a adicao de um item.
